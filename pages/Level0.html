SSH

SSH (Secure Shell) is a secure Network Protocol originally created for Unix systems to allow encrypted communication over a network.
It is the direct, secure successor to Telnet.

Accessing a remote server running an SSH service is highly straightforward, typically requiring just a single terminal command:

ssh username@server_ip

Some other fun things you can do when logging in SSH, consist of knowing what a port is, and choosing a designated port.

By default SSH runs on port 22, but we can change it simply by using -p [portno]
