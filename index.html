What exactly is OverTheWire: Bandit?

OverTheWire: Bandit is a popular, text-based wargame specifically designed for beginners to learn and practice the basics of security and Linux command-line operations.

Players progress through a series of levels by using terminal commands to find hidden passwords, which are then used to log into the next, increasingly difficult level.

It is widely considered an essential, hands-on starting point for anyone looking to get into cybersecurity, Linux system administration, or Capture The Flag (CTF) competitions.

So how to run linux without ruining your PC if you're not used to it?
Well there are many ways, but I suggest using a Virtual Machine (VM) .
What is a VM? Essentially it's a mini computer inside your computer!


To work inside a VM, you install a manager like Oracle VirtualBox and set it up there.

There are a few manager options depending on your computer:

  • Oracle VirtualBox: Completely free, open-source, and works perfectly on Windows and Mac. The best starting point!
    (This is what I'll be using)
  • VMware Workstation / Fusion: Industry-standard, super fast, and now completely free for personal use.
  • UTM: The absolute best, lightweight choice if you are using a modern Apple Silicon Mac.
